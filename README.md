# WebDAVProbe
WebDAVProbe — IIS fingerprinting &amp; WebDAV active exploitation testing tool. Detects WebDAV misconfigurations, enumerates allowed HTTP methods, and actually tests if they work (not just advertised). Maps IIS versions, tests PUT/DELETE/MOVE/COPY/MKCOL/PROPFIND/TRACE with real payloads, distinguishes auth-required vs blocked vs exploitable.
